# Autonomous_Agent_Forge_For_Calculation
An interactive Agentic AI workflow platform with GA(Genetic Algorithms) that shows how agents improve through prompt evolution, smart routing, and runtime tool generation. Users can submit problems, watch the backend process them live, and see how models, tools, and workflows adapt in real time to produce answers.
